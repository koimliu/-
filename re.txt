# Python `re` 常用正则表达式速查表

## 一、数字相关
| 需求         | 正则           | 示例            |
| ------------ | -------------- | --------------- |
| 匹配整数     | `^-?\d+$`      | `123`，`-456`   |
| 匹配小数     | `^-?\d+\.\d+$` | `3.14`，`-0.99` |
| 匹配任意数字 | `\d+`          | `123`           |

---

## 二、字母与单词
| 需求               | 正则        | 示例      |
| ------------------ | ----------- | --------- |
| 匹配英文单词       | `[A-Za-z]+` | `hello`   |
| 匹配字母数字下划线 | `\w+`       | `abc_123` |
| 匹配大写字母       | `[A-Z]+`    | `ABC`     |
| 匹配小写字母       | `[a-z]+`    | `abc`     |

---

## 三、联系方式
| 需求               | 正则                                          | 示例               |
| ------------------ | --------------------------------------------- | ------------------ |
| 手机号（中国大陆） | `^1[3-9]\d{9}$`                               | `13812345678`      |
| 电话号码（座机）   | `^\d{3,4}-\d{7,8}$`                           | `0531-12345678`    |
| 邮箱               | `[a-zA-Z0-9._%+-]+@[a-zA-Z0-9.-]+\.[a-z]{2,}` | `abc_123@test.com` |

---

## 四、证件号码
| 需求                 | 正则                      | 示例                 |
| -------------------- | ------------------------- | -------------------- |
| 身份证（15位或18位） | `^\d{15}$|^\d{17}[\dXx]$` | `370982199901011234` |
| 护照                 | `^[a-zA-Z0-9]{5,17}$`     | `G1234567`           |

---

## 五、网络相关
| 需求 | 正则                           | 示例                 |
| ---- | ------------------------------ | -------------------- |
| URL  | `https?://[^\s]+`              | `https://openai.com` |
| 域名 | `[a-zA-Z0-9.-]+\.[a-zA-Z]{2,}` | `www.test.com`       |
| IPv4 | `^(?:\d{1,3}\.){3}\d{1,3}$`    | `192.168.1.1`        |

---

## 六、日期与时间
| 需求               | 正则                  | 示例         |
| ------------------ | --------------------- | ------------ |
| 日期（YYYY-MM-DD） | `^\d{4}-\d{2}-\d{2}$` | `2025-08-28` |
| 时间（HH:MM:SS）   | `^\d{2}:\d{2}:\d{2}$` | `12:30:45`   |

---

## 七、实用替换示例
| 需求               | 正则与替换                                       | 示例                         |
| ------------------ | ------------------------------------------------ | ---------------------------- |
| 隐藏手机号中间四位 | `re.sub(r"(\d{3})\d{4}(\d{4})", r"\1****\2", s)` | `138****5678`                |
| 提取数字           | `re.findall(r"\d+", s)`                          | `['123','456']`              |
| 提取分组           | `(\d{4})-(\d{2})-(\d{2})`                        | `2025-08-28 -> (2025,08,28)` |
